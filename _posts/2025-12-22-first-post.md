## The Intersection Between Computing and Astronomy

Modern astronomy would be impossible without computational power. From processing terabytes of data to complex simulations of the universe, computing is essential.

## Tools I Use

### Python for Astronomy

```python
import numpy as np
import matplotlib.pyplot as plt
from astropy.io import fits

# Data analysis example
def analyze_fits_image(file):
    with fits.open(file) as hdul:
        data = hdul[0].data
        return np.mean(data), np.std(data)
```
