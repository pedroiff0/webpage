## About TOPCAT

TOPCAT (Tool for OPerations on Catalogues And Tables) is an interactive graphical viewer for tabular data.

## Key Features

- **Catalog Inspection**: Browse and inspect astronomical catalogs
- **Cross-Matching**: Match sources across multiple catalogs
- **Visualization**: Scatter plots, 3D plots, all-sky maps
- **Filtering**: Filter and subset catalog data
- **SQL Support**: Query data using SQL-like syntax

## Installation

```bash
# Download from official site
# Requires Java 8 or higher
java -jar topcat-full.jar
```

## Common Workflows

### Loading Catalogs

1. File → Open Table
2. Select FITS or VOTable
3. Browse and explore data

### Cross-Matching

1. Tools → Cross-Match
2. Select primary and secondary catalogs
3. Set matching radius
4. Review results

- [User Forum](http://www.star.bris.ac.uk/mbt/topcat/)

## Useful Features

- VO (Virtual Observatory) support
- Catalog download tools
- Advanced plotting capabilities
- Astronomical coordinate support

## Sobre TOPCAT

TOPCAT (Tool for OPerations on Catalogues And Tables) é um visualizador gráfico interativo para dados tabulares.

## Características Principais

- **Inspeção de Catálogos**: Navegue e inspecione catálogos astronômicos
- **Cross-Matching**: Faça corresponder fontes em múltiplos catálogos
- **Visualização**: Scatter plots, gráficos 3D, mapas all-sky
- **Filtragem**: Filtre e subdivida dados de catálogo
- **Suporte SQL**: Consulte dados usando sintaxe semelhante a SQL

## Instalação

```bash
# Baixe do site oficial
# Requer Java 8 ou superior
java -jar topcat-full.jar
```

## Fluxos de Trabalho Comuns

### Carregando Catálogos

1. File → Open Table
2. Selecione FITS ou VOTable
3. Navegue e explore dados

### Cross-Matching

1. Tools → Cross-Match
2. Selecione catálogos primário e secundário
3. Configure raio de correspondência
4. Revise resultados
