## About DS9

SAOImageDS9 is a free image viewer and toolkit for multi-wavelength astronomy and data visualization.

## Key Features

- **FITS Viewer**: Display and analyze FITS images
- **Multi-wavelength**: Handle X-ray, optical, infrared data
- **Analysis Tools**: Photometry, imaging statistics
- **Region Support**: Create and manage astronomical regions
- **Scripting**: Automate with Tcl/JavaScript

## Installation

```bash
# macOS with Homebrew
brew install saoimageds9

# Linux
sudo apt-get install saods9

# Windows: Download from SAO
```

## Quick Start

1. Open DS9
2. File → Open → Select FITS image
3. Use Scale menu to adjust image display
4. Use Region menu to create measurement regions

## Sobre DS9

SAOImageDS9 é um visualizador de imagens gratuito e kit de ferramentas para astronomia multilongitude de onda e visualização de dados.

## Características Principais

- **Visualizador FITS**: Exiba e analise imagens FITS
- **Multilongitude de Onda**: Lide com dados de raios X, ópticos, infravermelhos
- **Ferramentas de Análise**: Fotometria, estatísticas de imagem
- **Suporte a Regiões**: Crie e gerencie regiões astronômicas
- **Scripts**: Automatize com Tcl/JavaScript

## Instalação

```bash
# macOS com Homebrew
brew install saoimageds9

# Linux
sudo apt-get install saods9

# Windows: Baixe do SAO
```

## Início Rápido

2. File → Open → Selecione imagem FITS
3. Use menu Scale para ajustar exibição
4. Use menu Region para criar regiões de medição
