## About Aladin Lite

Aladin Lite is a lightweight, web-based, image viewer and catalog browser developed by the CDS.

## Key Features

- **Multi-wavelength View**: Overlay surveys from X-ray to radio
- **Catalog Browser**: Access millions of astronomical objects
- **Interactive Maps**: Pan, zoom, and explore the sky
- **No Installation**: Works directly in your browser
- **Coordinates**: Search by name or coordinates

## Getting Started

1. Visit [https://aladin.cds.unistra.fr/AladinLite/](https://aladin.cds.unistra.fr/AladinLite/)
2. Use the search bar to find objects
3. Select surveys from the menu
4. Click objects to see details

## Available Surveys

- **Optical**: SDSS, 2MASS, PanSTARRS
- **Infrared**: Spitzer, Herschel
- **X-ray**: Chandra, XMM-Newton
- **Radio**: VLA, NVSS

## Features

- Footprint visualization
- Catalog overlays
- Cone search queries
- MOC (Multi-Order Coverage) support
- API for custom applications

## Resources

- [Official Site](https://aladin.cds.unistra.fr/AladinLite/)
- [Documentation](https://aladin.cds.unistra.fr/AladinLite/doc/)
- [FAQ](https://aladin.cds.unistra.fr/AladinLite/doc/FAQ.html)

## Tips

- Use keyboard shortcuts for navigation
- Right-click to access context menus
- Combine multiple catalogs for deeper analysis

## Sobre Aladin Lite

Aladin Lite é um visualizador de imagens leve baseado na web e navegador de catálogo desenvolvido pelo CDS.

## Características Principais

- **Visualização Multilongitude de Onda**: Sobreponha levantamentos de raios X a rádio
- **Navegador de Catálogos**: Acesse milhões de objetos astronômicos
- **Mapas Interativos**: Panorama, zoom e explore o céu
- **Sem Instalação**: Funciona diretamente no seu navegador
- **Coordenadas**: Pesquise por nome ou coordenadas

## Primeiros Passos

1. Visite [https://aladin.cds.unistra.fr/AladinLite/](https://aladin.cds.unistra.fr/AladinLite/)
2. Use a barra de pesquisa para encontrar objetos
3. Selecione levantamentos do menu
4. Clique em objetos para ver detalhes

## Levantamentos Disponíveis

- **Óptico**: SDSS, 2MASS, PanSTARRS
- **Infravermelho**: Spitzer, Herschel
- **Raios X**: Chandra, XMM-Newton
- **Rádio**: VLA, NVSS

## Características

- Visualização de pegadas
- Sobreposição de catálogos
- Consultas de busca cônica
- Suporte MOC (Multi-Order Coverage)
- API para aplicações personalizadas

## Recursos

- [Site Oficial](https://aladin.cds.unistra.fr/AladinLite/)
- [Documentação](https://aladin.cds.unistra.fr/AladinLite/doc/)
- [FAQ](https://aladin.cds.unistra.fr/AladinLite/doc/FAQ.html)

## Dicas

- Use atalhos de teclado para navegação
- Clique com o botão direito para acessar menus de contexto
- Combine múltiplos catálogos para análise mais profunda
